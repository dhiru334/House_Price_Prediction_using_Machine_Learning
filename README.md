# House Price Prediction using Machine Learning in Python

## Project Overview

This project focuses on predicting house prices using Machine Learning techniques in Python. The goal of the project is to analyze housing data, preprocess the dataset, perform exploratory data analysis (EDA), and build predictive machine learning models for estimating house prices.

The project demonstrates the complete machine learning workflow, including data cleaning, feature engineering, model training, evaluation, and prediction.

# Project Workflow

## 1. Data Collection

* Imported housing dataset into Jupyter Notebook.

## 2. Data Preprocessing

Performed several preprocessing steps:

* Checked missing values
* Handled null data
* Removed unnecessary columns
* Converted categorical variables
* Prepared dataset for machine learning models
